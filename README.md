
# Kwizzit

*Kwizzit a SwiftUI product.*  
Kwizzit is a Anime based quiz app. 


## Live Link
 *[Kwizzit](https://kwizzit.netlify.app/)*

## Features

- Home Page
- Quiz Category Page 
- Rules Page for Quiz
- Quiz Question Page
- Result Page with Answers & Score
- Login Page  
- Sign UP Page
- Quit Page
- Finish Quiz
